Available JSON API endpoints:

  • /pkg
  • /pkg/{package}/policy
  • /pkg/{package}/run
  • /pkg/{package}/run/{run_id}
  • /pkg/{package}/run/{run_id}/diff
  • /pkg/{package}/merge-proposals
  • /merge-proposals
  • /queue
  • /run
  • /run/{run_id}
  • /package-branch
  • /{suite}/published-packages
  • /{suite}/pkg/{package}/schedule
  • /{suite}/pkg/{package}/diff
  • /publish/{publish_id}
  • /policy
  • /runner/status
  • /active-runs
  • /active-runs/{id}/log
  • /active-runs/{id}/log/{logfile}
  • /{suite}/report

E.g. to schedule a new run of lintian-brush for cowsay: 

curl -X POST {{ url.join(URL('lintian-fixes/pkg/cowsay/schedule')) }}
